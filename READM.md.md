# Random Quote Generator (Генератор случайных цитат)

**Автор:** Алексей Смирнов

## Описание

Random Quote Generator — это десктопное приложение на Python с графическим интерфейсом (tkinter) для поиска вдохновения в мудрых изречениях. Программа позволяет генерировать случайные цитаты из встроенной коллекции, добавлять собственные цитаты, фильтровать по автору и теме, а также сохранять историю генераций.

## Функциональные возможности

- 🎲 **Генерация случайной цитаты** из обширной базы (включает 12 предустановленных цитат)
- ➕ **Добавление собственных цитат** с указанием текста, автора и темы
- 🔍 **Фильтрация** по автору и/или теме (влияет и на генерацию, и на отображение истории)
- 📜 **Просмотр истории** сгенерированных цитат в удобной таблице
- 💾 **Автоматическое сохранение** всех пользовательских цитат и истории в JSON-файлы
- 📋 **Копирование** текущей цитаты в буфер обмена
- 📤 **Экспорт истории** в текстовый файл
- 🧹 **Очистка истории** при необходимости
- 🎯 **Валидация** вводимых данных (проверка на пустые строки, дубликаты цитат)

## Требования

- Python 3.6 или выше
- Стандартная библиотека tkinter (входит в состав Python)
- Дополнительные модули не требуются
